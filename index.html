Pen: Pure CSS Slide Down Toggle

by Surjith

This pen creates a button which triggers an element to slide down onto the page, pushing the rest of the content down. By pressing the button again, that element will slide back up off the page and out of the viewport. It can be done with pure CSS (meaning no need for Javascript) and HTML.

This pen makes use of...

  • CSS transition
  • :checked pseudo class
  • ::after pseudo element
  • + and ~ selectors

See the Pen Pure CSS Slide Down Toggle by Surjith (@surjithctly) on CodePen.
